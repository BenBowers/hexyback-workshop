# Deployment

## Install dependencies

1. Open a __new__ terminal

2. In the __new__ terminal, run the following:

  ```shell
  pnpm i
  ```

## Deploy the application

1. In the same terminal, change into the `backend` directory

  ```shell
  cd backend
  ```

2. Run the deployment target

  ```shell
  pnpm run deploy
  ```

  You'll be prompted to a stage name. Enter your first name and hit `Enter ‚èé`:

  ```shell
  pnpm run deploy

  Please enter a name you'd like to use for your personal stage. Or hit enter to use node: joe 
  ```

## Run the tests

### Unit tests:

1. Open a __new__ terminal

2. In the __new__ terminal, run the following:

  ```shell
  pnpm run test
  ```

### Integration tests:

1. Open a __new__ terminal

2. In the __new__ terminal, run the following:

  ```shell
  pnpm run test:integration
  ```

## Start the documentation:

1. Open a __new__ terminal

2. In the __new__ terminal, run the following:

  ```shell
  pnpm run docs
  ```
